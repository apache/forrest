Supported Features

  • This presentation is intended to illustrate most of the features of Open Office Impress

  • It can be used to check which features are supported by SAAFE-D

  • If you find a feature is not in this presentation please feel free to add it

So far very few few features within Open Office are supported. This document is intended to help identify features that are and are not supported.

This presentation has slides that illustrate the many features of Open Office. Each slide describes the functionality being displayed in that slide. Thus users should be able to navigate through this presentation in order to see which features are supported.

If you see a feature is missing from this presentation please feel free to add it.
