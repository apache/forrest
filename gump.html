Apache Gump integration

Apache Gump builds Apache Forrest trunk each day. If dependencies (e.g. Cocoon and certain blocks) fail, then of course the build of Forrest will fail.

The Gump metadata descriptor for Forrest defines a number of different projects. Notifications are sent to the forrest-dev mailing list.
