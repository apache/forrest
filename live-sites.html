If you want your site added to the list, then please send a request to the mailing lists or to the Issue tracker. Tell us the URL, Name, and catch-phrase.

Sites using upcoming Dispatcher

The skinning system in versions of Forrest up to and including 0.7 was great, however, it proved to be too inflexible. Soon we will have brand new system called Dispatcher. The following sites are examples.

Note
The Dispatcher is still in development (as of 0.8-dev cycle), please only use them if you know what you are doing. We expect to include them in an upcoming release at which point we will be encouraging their use.
  • Floating Sun - Diwaker Gupta :: Web/Blog/Gallery
  • Cyriaque family website.
  • Apache Lenya Zone - this site is using v2 with some nice demonstrations of new features like the theme switcher.
  • tarGET-X - know the unknown by thorsten.

  • varietee™ adjustable golf practice platforms
  • Simal - an RDF/XML project registry framework.
  • jrds - a cacti like tools written in java
