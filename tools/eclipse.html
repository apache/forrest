Eclipse Plugin

The Forrest plugin for Eclipse is new and under rapid development. See the

Note
You need to separately download the activation.jar (its licensing means that we cannot distribute it).

Get it from http://java.sun.com/products/javabeans/glasgow/jaf.html - Download the JAF 1.0.2 ZIP, extract activation.jar, and copy it to tools/eclipse/plugins/org.apache.forrest.eclipse.servletEngine/lib/ directory.

Installation notes
