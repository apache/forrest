Overview

This plugin will allow to read site content by computer voice and navigate throug it via speech. At this early stage of developement, this plugin will work only for very simple site with non-structured sections.

Requirements

You should have borwser which supports X+V technology. I can recommend Opera, I think it's the best solution for now. After you download it you should enable voice in options by selecting "tools -> preferences -> advanced tab -> voice (left navigation) -> check Enable voice options", this will download voice libraries. Then you are able to process .mxml files

Another important thing is, that you can use this plugin only in "run" mode.

Usage

Currently, there is no visual output at all. After loading .mxml file the browser will start reading the first section. Try it by reading this file.

You can interrupt reading by pressing "Print Screen" key. Once you have interupted reading you can issue a voice command as follows:

  • go to next - skip to next section
  • go back - skip to previos section
  • a number - skip to section with specific number

If you don't interrupt voice output, the reading will automatically continue to next section after 2 seconds. If you say a bad number (out of range), you will be prompted for correct one.

That is all for now...
