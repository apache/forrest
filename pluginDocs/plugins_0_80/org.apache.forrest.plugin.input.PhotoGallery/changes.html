Introduction and explanation of symbols

Version 0.3 (not-released)

code

  • Added a gallery/index page that lists the multiple galleries. Enabled albums to contain metadata. Automated the image variant (e.g. thumb, small) creation. Note that this is not backward-compatible, so existing users need to move original images from the old "big" directory and remove the "preview" and "small" directories. Committed by TWW.
  • Enable "albuminfo" to contain links and other markup. Committed by DC.
  • Use the "albuminfo" to set the title and description of each gallery. Committed by DC.
