Version 0.2-dev (not-released)

Changes to the Code Base

  • Width of output text is now configurable from forrest.properties.xml using the property name text.page.width.

    Committed by RDG.
  • Use locationmap for resolution of resources and files. Resources are exposed via {lm:text.*} Committed by RDG.

Version 0.1 (not-released)

Changes to Project Administration

  • Changed name to bring into line with new naming conventions. Committed by RDG.

Changes to the Code Base

  • Created the initial plugin based on Ross Gardler's document2txt.xsl stylesheet. Committed by RFT.
  • Initial cut of the common stylesheet which contains text functions that currently handles newline, carriage returns, underlines, borders. Committed by RFT.
  • Preliminary handling of section indenting, section title underlining, note/warning/fixme elements and list handling. Committed by RFT.
  • Implemented text wrapping Committed by RFT.
  • Implemented nested elements such as lists inside lists, tables inside tables, lists inside tables, etc. Committed by RFT.
  • Table handling Committed by RFT.
  • Hanging indents on ordered, unordered and definition lists Committed by RFT.

doc

  • Updated the plugin documentation with features and limitations. Committed by RFT.
