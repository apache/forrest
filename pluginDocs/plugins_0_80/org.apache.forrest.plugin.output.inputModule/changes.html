Version 0.1 (not-released)

Changes to the Code Base

  • Initial plugin code. The heart of this plugin is the ModuleGenerator. The generator is contacting the input module and gets an iterator of all attributes names. This iterator will be then used to receive the values of the key. Till now we only use a Object to String cast but an input module does not have to return a simple String but can return any object. This may cause ClassCastExceptions for the values. Committed by TS.
