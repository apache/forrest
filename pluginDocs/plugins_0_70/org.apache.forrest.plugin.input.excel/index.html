Overview: Excel Table Input Plugin

This is documentation for current version v0.7 (More)
  • Summary
  • Documentation
  • Examples

Summary

With the Excel table input plugin it is possible to create forrest tables directly from Excel XML output. Excel sheets will be translated into sections, and each sheet will generate a table. Only simple tables are supported (by now). Formatting is not transferred to forrest but some calculations are.

Needs Excel 2003 to save the content as XML.

Documentation

The How-to section describes step by step how to create and include an Excel table in forrest.

For implemenation details see resources/stylesheets/table2document.xsl.

Examples

The samples section contains the following examples:

  • Simple table: Shows the basic features.
  • Table with figure: Shows how to embedd a figure with the table plus additional text.
  • Color table: Specialized table to highlight cells in the color specified.
