Changes to Project Administration

  • Changed name to bring into line with new naming conventions. (RDG)

Changes to the Code Base

  • Created the initial plugin based on Ross Gardler's document2txt.xsl stylesheet. (RFT)
  • Initial cut of the common stylesheet which contains text functions that currently handles newline, carriage returns, underlines, borders. (RFT)
  • Preliminary handling of section indenting, section title underlining, note/warning/fixme elements and list handling. (RFT)
  • Implemented text wrapping (RFT)
  • Implemented nested elements such as lists inside lists, tables inside tables, lists inside tables, etc. (RFT)
  • Table handling (RFT)
  • Hanging indents on ordered, unordered and definition lists (RFT)
