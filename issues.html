Apache Forrest issue tracker

The Apache Forrest uses an issue tracker call Jira.

http://issues.apache.org/jira/browse/FOR

You need to set up an account there before you can create new issues or add comments to existing issues.

Note that the issue tracker is not a discussion or user help forum. Please discuss issues with using Forrest on the user or dev mailing list.
