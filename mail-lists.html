Important Notices

Before posting a question or problem to any mailing list, please first look at the following resources:

  • Forrest FAQs
  • Forrest documentation
  • Cocoon Links for pointers to general XML and XSL and Java related resources. Please do not ask such questions here unless they are related to Forrest.
  • Mailing list archives - a veritable goldmine of Forrest-specific information.
  • Contribution notes and tips.

If you are posting about a problem that you are having, it will aid in finding a speedy resolution if you provide full configuration details, including the version number of Forrest, your operating system, JDK version, and servlet engine, and full details of any errors encountered including full error messages and stack traces.

Forrest User

Purpose: Any general issue related to the day-to-day use of Forrest.

Note: This is a subscriber-only list (in an effort to stop spam), so you must join before you can post.

Subscribe: user-subscribe<at>forrest.apache.org

Un-subscribe: user-unsubscribe<at>forrest.apache.org

Help: user-help<at>forrest.apache.org

Forrest Dev

Purpose: Any issue related to development and enhancement.

Note: This is a subscriber-only list (in an effort to stop spam), so you must join before you can post.

Subscribe: dev-subscribe<at>forrest.apache.org

Un-subscribe: dev-unsubscribe<at>forrest.apache.org

Help: dev-help<at>forrest.apache.org

Read-only NNTP access to forrest-dev is available through the gmane.text.xml.forrest.devel newsgroup

Note
If sending code patches, then please add [PATCH] to the email Subject line and use ./build.sh patch (or build patch in Windows) to generate a patch file. However, it is preferable to send patches via our Jira issue tracker.

Forrest SVN

Purpose: Receive notifications on changes in the Forrest codebase in the SVN repository.

Subscribe: svn-subscribe<at>forrest.apache.org

Un-subscribe: svn-unsubscribe<at>forrest.apache.org

Help: svn-help<at>forrest.apache.org
